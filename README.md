# 마크다운

- 문서의 구조, 내용 작성
- 마크다운을 이용한 블로그 운영
- e.g) Jupyter Notebook, Notion, Typora

---

### #사용하기(헤딩)

* 문서의 제목이나 소제목으로 사용
* #의 개수에 따라 제목의 수준 구별
* 글자 크기 키우기 위해서 사용x



### 리스트 

#### 순서가 있는 리스트, 순서가 없는 리스트

* 목록을 표시하기 위해 사용
* *,- 사용해서 순서가 없는 리스트 만들기 가능

1. 순서가있는 리스트
   1. 탭을 사용하면 순서가 있는 하위 리스트 만들기 가능

2. 2번 리스트

엔터두번 치면 리스트 사라짐



### 코드 블럭

* 코드를 이쁘게 출력
* 사용하는 프로그램에 따라 특정 언어를 명시하면 Syntax Highlighting 지원

```python
a = 25

if a >= 20
	print("성인입니다")
```



### 링크 

* 다른 페이지로 이동하는 링크를 삽입

[string] (url) 형식으로 표현

![string] (img_url) 링크와 비슷하지만 이미지를 삽입



### 텍스트 강조 

* 굵게, 기울임, 취소선을 이용해 텍스트 강조

#### Bold 

** 굵게 **

#### Italic 

_ 기울임 _

#### Strikeout

~~ 취소선 ~~

* *를 _로 대체 가능



### 수평선 

* 가로로 긴 수평선 작성
* 단락 구분할 때 사용

--- 으로 사용

* -(hypen)을 * 또는 _(underbar)로 대체 가능
